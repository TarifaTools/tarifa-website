Build multiple apps

Produce unique apps for multiple platforms with a single project and discover the simplest way to build white-label apps with Cordova.

Integrate your front-end build system

Integrate the tools you use to build and watch your www project and tarifa will call them.

Interactive project bootstrap

No need to remember everything while creating a project, tarifa will guide you and save your answers in the project files. Cordova plugin bootstrap is also in the box.

Deploy to testers from the CLI

No need to launch a browser to upload binaries to hockeyapp.

Manage iOS provisioning files

No need to use the apple provisioning portal each time you add a new device to a provisioning file.

Available platforms

tarifa supports a subset of Cordova platforms: browser, ios, android and windows phone 8.
