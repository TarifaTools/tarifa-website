Build multiple apps

Produce multiple unique apps easily on a given platform within a single project.

Integrate your front-end build system

Integrate the front-end build tool you use and tarifa will trigger it.

Interactive project bootstrap

No need to remember every required information while creating a project, tarifa will guide you with adequate questions and memorize the answers in the project files.

Deploy to testers from the CLI

No need to launch a browser to upload binaries to hockeyapp.

Manage iOS provisioning files

No need to use the provisioning portal each time you add a new device to a provisioning file.

Available platforms

tarifa supports a subset of cordova platfoms: web, ios, android, windows phone 8.
